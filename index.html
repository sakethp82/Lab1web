Delivering Custom Fonts via Web Server

What is Font Delivery?

When a browser loads a webpage, it typically uses fonts installed on the user's device. However, if a web designer wants to use a custom font that isn't installed on most systems, it must be delivered to the client via a web server.

How to Serve a Custom Font

One of the most effective methods for delivering a custom font is by using the @font-face rule in CSS. The font file (typically in WOFF or WOFF2 format) is stored on the server, and the CSS references the file so that the browser can download and apply it.

Steps to Implement @font-face

1. Choose a custom font and download it in WOFF/WOFF2 format.
2. Upload the font file to your web server.
3. Use the @font-face rule in your CSS to define the font source.
4. Apply the font-family property to your text elements.
